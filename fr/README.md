# Parcours francais IOAI 2024

Cette arborescence `fr/` est la version francaise miroir du depot. L'objectif est qu'un eleve francophone puisse suivre les taches avec le meme niveau de comprehension que dans la version anglaise.

## Taches disponibles

| Tache | Notebook francais | Etat |
|---|---|---|
| Help BOBAI | [Notebook](On-Site-Round/Help_BOBAI/Help_BOBAI.ipynb) | Verifie |
| Lost in a Hyperspace | [Notebook](On-Site-Round/Lost_in_Hyperspace/Lost_in_Hyperspace.ipynb) | Verifie |
| The Madarian Cow Mystery | [Notebook](On-Site-Round/Madarian_Cow/Madarian_Cow.ipynb) | Verifie |

## Regle de fidelite

Les notebooks francais conservent la meme structure, le meme ordre de cellules, le meme code executable et les memes donnees que la version originale. Les cellules Markdown, commentaires et instructions destines aux lecteurs ont ete traduits en francais, puis verifies contre les notebooks sources.
