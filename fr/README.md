# Parcours francais IOAI 2024

Cette arborescence `fr/` est destinee a devenir la version francaise miroir complete du depot. L'objectif est qu'un eleve francophone puisse suivre les taches et les solutions avec le meme niveau de comprehension que dans la version anglaise.

## Etat reel de la traduction

Les enonces francais sont disponibles, mais la traduction haute fidelite du depot n'est pas encore terminee tant que les notebooks de solution ne sont pas eux aussi traduits et verifies.

| Tache | Enonce francais | Solution francaise | Etat |
|---|---|---|---|
| Help BOBAI | [Notebook](On-Site-Round/Help_BOBAI/Help_BOBAI.ipynb) | `On-Site-Round/Help_BOBAI/Solution/Help_BOBAI_Solution.ipynb` | En cours : solution a traduire |
| Lost in a Hyperspace | [Notebook](On-Site-Round/Lost_in_Hyperspace/Lost_in_Hyperspace.ipynb) | `On-Site-Round/Lost_in_Hyperspace/Solution/Lost_in_Hyperspace_Solution.ipynb` | En cours : solution a traduire |
| The Madarian Cow Mystery | [Notebook](On-Site-Round/Madarian_Cow/Madarian_Cow.ipynb) | `On-Site-Round/Madarian_Cow/Solution/Madarian_Cow_Solution.ipynb` | En cours : solution a traduire |

## Regle de fidelite

Les notebooks francais doivent conserver la meme structure, le meme ordre de cellules, le meme code executable, les memes donnees et les memes sorties attendues que la version originale. Seules les cellules Markdown, les commentaires et les instructions destinees aux lecteurs doivent etre traduits en francais, avec verification explicite avant de marquer une tache comme terminee.
